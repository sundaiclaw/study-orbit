# Proposal: Study Orbit

## Why
Learning dashboards are usually static trackers. The wireframe suggests a cleaner experience: one progress anchor and two focused cards. This is a good fit for an AI coach that helps a user resume the right lesson and know what comes next.

## What changes
Build a web app where the user enters a learning goal and available study time, then an AI coach generates:
- a short study plan
- a current lesson card
- a next lesson card
- daily motivation and checkpoint guidance
- a visual progress ring

## Success criteria
- User can enter a goal and time budget.
- AI returns structured lesson-plan JSON.
- UI renders progress plus two key lesson cards matching the sketch-inspired layout.
- App is demoable instantly with a default goal.
