# Spec: AI learning dashboard

## Requirements

### Requirement: study plan generation
The system shall generate a structured study plan from a user goal.

### Requirement: focused dashboard
The system shall display progress and two prioritized lesson cards.

### Requirement: coach guidance
The system shall provide motivational and tactical AI guidance in the dashboard.

### Requirement: instant demo
The system shall ship with a default topic so reviewers can test without setup.
